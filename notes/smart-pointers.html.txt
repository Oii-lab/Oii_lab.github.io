        <h1>智慧指標 (Smart Pointers) 原理詳解</h1>
        <p class="tag">C++11</p>
        <p>深入理解 RAII 觀念與 unique_ptr、shared_ptr 的底層實現，是掌握現代 C++ 記憶體管理的核心。</p>

        <h2>1. RAII 原則</h2>
        <p>RAII (Resource Acquisition Is Initialization) 是 C++ 中管理資源的關鍵技術，智慧指標就是基於此原則的經典實作。</p>

        <h2>2. unique_ptr 範例</h2>
        <pre class="line-numbers"><code class="language-cpp">
#include <memory>
#include <iostream>

void example_unique_ptr() {
    // 資源獲取：unique_ptr 接收 new 運算子的結果
    std::unique_ptr<int> ptr = std::make_unique<int>(100);

    // ... 進行操作 ...

    // 離開作用域時，資源會自動釋放 (delete)
    std::cout << "ptr 將在離開作用域後自動釋放記憶體" << std::endl;
} // <--- 記憶體在這裡被安全釋放

int main() {
    example_unique_ptr();
    return 0;
}
        </code></pre>
